# Descrição do Projeto

Considere um **sistema de controle automatizado** para uma rede de estacionamentos rotativos. Neste sistema o cliente paga uma <u>mensalidade</u> e tem acesso livre em qualquer unidade de estacionamento da rede. 

O sistema também disponibiliza um <u>umapa</u> com rotas para as unidades mais próximas. 

## Documentos

- Diagrama de classes (Cenários abaixo):
- Desenvolva considerando a categorização BCE;
- Represente a integração com uma API de pagamento;
- Desconsidere métodos getters e setters.
