# User Stories - Sistema de eventos

## Participante

- Como participante
  Eu quero me cadastrar no sistema para participar do evento;

- Como participante
  Eu quero me inscrever no evento para garantir minha participação;

- Como participanete
  Eu quero visualizar a programação do evento para escolher atividades de interesse;

- Como participante
  Eu quero me inscrever em atividades especificas para reservar minha vaga;

- Como participante
  Eu quero cancelar minha inscrição em uma atividade caso não possa participar;

## Organizador

- Como organizador
  Eu quero cadastrar atividades para compor a programação do evento;

- Como organizador, quero visualizar a lista de participantes inscritos em cada atividade para organização do evento;

- Como organizador, quero definir limite de vagas para cada atividade para controlar a capacidade;
