# Fluxo Principal - Realizar Compra

1. O cliente visualiza o catálogo; 
2. O cliente seleciona o produto para adicionar ao carrinho; 
3. O sistema atualiza o carrinho com o novo produto; 
4. O cliente seleciona o carrinho; 
5. O sistema exibe os produtos adicionados ao carrinho;
6. O cliente seleciona "Finalizar Compra";
7. O sistema solicita dados do usuário;
8. O cliente fornece endereço de entrega;
9. O sistema calcula o frete e exibe o valor total;
10. O sistema solicita que o cliente escolha a forma de pagamento;
11. O cliente seleciona a dorma de pagamento no PIX;
12. O sistema gera o QRcode para pagamento no PIX;
13. O sistema verifica pagamento;
14. O sistema exibe mensagem "Pagamento Efetuado";
15. O sistema exibe um resumo da compra e da entrega;
16. Caso de uso encerrado;
